ECLIPSE

TIPS

Cuando editamos código Java con tabulaciones adaptadas por nosotros y aplicamos el formateo automático,
se pierden nuestros cambios. Para evitar esta situación, hay que configurar Eclipse en propiedades de nuestro workspace:


1. Preference.Java.Code Style.Formatter y en la sección

	Line Wrapping   
	
		X Never join already wrapped lines (Marcar esta opción)


2. Generar documentación clases Java

JAutoDoc

http://jautodoc.sourceforge.net/update/
