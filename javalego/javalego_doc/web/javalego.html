JAVALEGO ¿QUÉ ES? Es una Arquitectura Colaborativa Open Source que permite construir de productos software para distintas áreas tecnológicas. Antecedentes ¿Cansado de aprender frameworks, librerías, lenguajes, ...? ¿Cansado de su baja productividad por motivos tecnológicos? ¿Cansado de desarrollar en tecnologías obsoletas? ¿Cansado de realizar migraciones hacia nuevas tecnologías? Nuestros objetivos Desarrollar aplicaciones con código simple y claro que ejecute funcionalidades concretas. Desarrollar código que se ejecute en diferentes plataformas (web, híbrida o nativa para móviles). Desarrollar código que integre frameworks y librerías de referencia demandadas en nuestros proyectos. Desarrollar código que pueda evolucionar hacia nuevas tecnologías emergentes sin realizar cambios. Mejorar nuestra productividad para ser más competitivos y felices desarrollando. IMAGEN CAMINO CON BROZAS O DESPEJADO. ¿Cómo? "Arquitectura de productos" Hemos desarrollado una arquitectura colaborativa open source basada en Java que permite construir productos software de calidad para resolver requerimientos funcionales concretos. "Un código base soportado para múltiples tecnologías" La idea es crear una especificación simple y clara que nos permita desacoplarnos de la tecnología real utilizada. A partir de esta especificación, construiremos los productos que implementen esta especificación acoplados con las tecnologías requeridas en nuestros proyectos. Esta versatilidad, permitirá reutilizar estos productos en otros proyectos que demande esta funcionalidad y tecnología. "Compartir el conocimiento" La experiencia adquirida en el uso de frameworks, librerías y plataformas servirá para compartir su conocimiento. "Publicar nuestros productos" Para poder registrar y publicar nuestros productos, acceda a nuestra Tienda virtual. "Calidad del software" Para garantizar la calidad de los productos registrados en la Tienda, se han definido unas mínimas reglas de calidad a cumplir y accesibles.
