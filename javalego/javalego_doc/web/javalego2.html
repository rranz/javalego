JAVALEGO es código abierto y se distribuye bajo licencia Apache 2. Normativa de calidad. Certificación aplicada a productos ayudan a la certificación de proyectos ¿Cómo funciona JAVALEGO? Desarrollamos componentes y productos de arquitectura y negocio para desarrollar aplicaciones empresariales Desarrollamos aplicaciones multiplataforma a partir de un modelo de negocio codificado en lenguaje Java Diagrama. Imagen del proceso: modelo -> gear arquitectura javalego -> plataformas Componentes Para desarrollar nuestro modelo de negocio, usaremos un conjunto de “componentes” que cubrirán nuestros requerimientos funcionales típicos de cualquier aplicación (UI, seguridad, acceso a datos, servicios, reglas, informes, …) Los componentes constituyen la especificación de las características de sus productos derivados o implementaciones. Características: Basado en interfaces Codificación simple Fácilmente extensible Débilmente acoplados para facilitar su integración en cualquier proyecto Desacoplados de frameworks y librerías para permitir su implementación en múltiples tecnologías Organizados por categorías JAVALEGO ofrece un conjunto de componentes Open Source que cubren muchas de las funcionalidades comunes de cualquier aplicación Productos La implementación de la especificación de componentes estará representada en la arquitectura por “productos” Características: Los productos serán las piezas reutilizables que integraremos en nuestros proyectos para desarrollar un requerimiento funcional Los productos podrán requerir o no de frameworks y librerías de referencia en el mundo Java (Spring, Vaadin, Shiro, Eclipse Birt, JSF, …) para obtener sus funcionalidades Podrán implementarse para diferentes plataformas: web o aplicaciones nativas para dispositivos móviles Los productos podrán ser de arquitectura o de negocio Los productos estarán desacoplados para poder integrarse fácilmente en cualquier proyecto. Queremos que sean productos no intrusivos que ayuden al desarrollo, no imponer el uso de una arquitectura completa Los productos publicados tendrán que ofrecer garantías de calidad mediante: documentación, usabilidad, cobertura de código y cobertura de test JAVALEGO aporta productos Open Source para cubrir gran parte de las funcionalidades comunes de cualquier aplicación JAVALEGO aporta productos para desarrollar aplicaciones en Vaadin y Android Puede colaborar a nuestra comunidad desarrollando productos para nuevas plataformas, frameworks o librerías.
