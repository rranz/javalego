ARQUITECTURA:

	Concepto:
	MeccanoJ ha desarrollado una arquitectura Open Source basada en componentes que permite desarrollar aplicaciones web y aplicaciones nativas para móviles usando Java o Groovy como lenguaje para definir su modelo de negocio.
